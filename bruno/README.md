## 📁 Location

All Bruno tests are in: `bruno/` directory

## 🚀 Quick Start

1. **Open Bruno Application**
   - Download from https://www.usebruno.com/

2. **Import Collection**
   - File → Open Collection
   - Navigate to `bruno/` folder

3. **Select Environment**
   - Choose "production" environment

4. **Run Tests**
   - Click "Run All Tests"
   - Or run individual `.bru` files
